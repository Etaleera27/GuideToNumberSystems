Conversion for Binary

Binary to Decimal

11011001₂ to Decimal (Using Method 1: Division/Multiplication)

Step 1: Write down place values

Binary digits (bits) from right to left have place values of powers of 2.

Step 2: Multiply each bit by its place value

  • 1 × 2⁷ = 1 × 128 = 128
  • 1 × 2⁶ = 1 × 64 = 64
  • 0 × 2⁵ = 0 × 32 = 0
  • 1 × 2⁴ = 1 × 16 = 16
  • 1 × 2³ = 1 × 8 = 8
  • 0 × 2² = 0 × 4 = 0
  • 0 × 2¹ = 0 × 2 = 0
  • 1 × 2⁰ = 1 × 1 = 1

Step 3: Add them up

128 + 64 + 0 + 16 + 8 + 0 + 0 + 1 = 217
Answer: 11011001₂ = 217₁₀

Binary to Octal

11011001₂ to Octal (Using Method 2: Substitution/Grouping)

Step 1: Group binary digits in sets of 3 from right

Binary: 11011001 → group in 3s → 11 011 001
Pad the leftmost group with zeros → 011 011 001

Step 2: Convert each group to octal

  • 011 → 3
  • 011 → 3
  • 001 → 1

Step 3: Combine the digits

Octal = 331₈
Answer: 11011001₂ = 331₈

Binary to Hexadecimal

11011001₂ to Hexadecimal (Using Method 2: Substitution/Grouping)

Step 1: Group binary digits in sets of 4 from right

Binary: 11011001 → groups in 4s 1101 1001

Step 2: Convert each group by building the value

We use the binary place values 8, 4, 2, 1 directly:
First group:1101

  • 1×8 = 8
  • 1×4 = 4
  • 0×2 = 0
  • 1×1 = 1

Add inside the group:
8 + 4 + 1 = 13

Now count past 9 in hexadecimal:
10 = A
11 = B
12 = C
13 = D

So: 1101 → D

Second group: 1001

  • 1×8 = 8
  • 0×4 = 0
  • 0×2 = 0
  • 1×1 = 1

Add:
8 + 1 = 9

So: 1001 → 9

Step 3: Combine the digits

Hexadecimal = D9₁₆
Answer: 11011001₂ = D9₁₆
